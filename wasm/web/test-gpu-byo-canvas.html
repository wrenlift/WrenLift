Smoke test — bring-your-own canvas

Two canvases the page already owns (declared inline in the HTML below). Wren attaches to each by id via Window.create({"canvas": "<id>"}), then paints them with different solid colours from the same Wren program. Demonstrates the multi-canvas path: each call mints a distinct registry handle and feeds an independent device.createSurface.
